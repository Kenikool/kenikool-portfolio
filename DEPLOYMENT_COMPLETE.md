## ğŸ“ Step-by-Step Guide

### Before You Start

1. **Create GitHub Account** (if you don't have one)

   - Go to https://github.com/signup
   - Follow the registration process

2. **Install Git** (if not installed)
   - Download from https://git-scm.com/download/win
   - Install with default settings

### Deployment Steps

#### Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. **Repository name:** `kenikool-portfolio` (or `your-username.github.io`)
3. **Visibility:** Public
4. **Don't** initialize with README
5. Click **"Create repository"**

#### Step 2: Deploy Your Portfolio

**Option A: Use the Script**

- Double-click `deploy-to-github.bat` in the portfolio folder
- Enter your repository URL: `https://github.com/YOUR-USERNAME/kenikool-portfolio.git`
- Press Enter and wait for completion

**Option B: Manual Deployment**

```bash
cd portfolio
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR-USERNAME/kenikool-portfolio.git
git branch -M main
git push -u origin main
```

#### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **"Settings"** tab
3. Click **"Pages"** in the left sidebar
4. Under **"Source"**:
   - Branch: **main**
   - Folder: **/ (root)**
5. Click **"Save"**

#### Step 4: Access Your Live Site

Wait 2-5 minutes, then visit:

- `https://YOUR-USERNAME.github.io/kenikool-portfolio`
